# Quik

* ## O que é o framework escolhido

    Qwik é um novo framework, criado pela Buider.io, cujo os principais objetivos são:

    * Perfomance
    * Interatividade

* ## Por que utilizá-lo
    O Qwik restaura a aplicação apartir de um ponto especifico, apenas executando o codigo necessario

* ## Em quais cenários ele não é indicado
    * Necessidade de utilizar bibliotecas especificas, como:
        * Gráficos complexos, 
        * Mapas, 
        * editores de texto rico
    * Curva de Aprendizado e Prazos Curtos
        Se sua equipe domina React ou Vue e o prazo está apertado, a "mentalidade Qwik" pode gerar bugs difíceis de depurar no início.

# Instalação

* ## Requisitos do ambiente
    Os pré-requisitos para a instalação do framework são:
    * .Node.js que seja v18.17 ou melhor
    * .IDE

* ## Comando de criação do projeto

    Agora deve ser escrito no terminal de seu IDE o seguinte comando:
    - npm create qwik@latest

    Então você deverá selecionar um dos ambientes a seguir:
    * Empty App (Qwik City + Qwik):
        .O melhor ponto de partida para um novo site (inclui rotas, layouts, etc.).

    * Library (Qwik):
        .Use esta opção se estiver criando uma biblioteca de componentes reutilizáveis, e não um site.

    * Playground App (Qwik City + Qwik):
        Um aplicativo pré-configurado com exemplos para explorar os recursos do Qwik.

    * E2e Library:
        Específica para projetos de testes de ponta a ponta.
